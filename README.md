# covid-19-graphs-jo
COVID-19 graph - especially cases per capita and growth.

## Run
```
git clone https://github.com/jojo4u/covid-19-graphs-jo.git
cd covid-19-graphs-jo
git submodule init
git submodule update 
python covid-19-graphs-jo.py
```

## For data update:
```
cd covid-19-data
python process.py
```

## Requirements:
* matplotlib
* numpy
* pandas
* seaborn
